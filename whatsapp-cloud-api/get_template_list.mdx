title: "GET Templates List"
description: "API to get list of all the templates with their current status"
---

#### **API End Point**

```
https://api.v7. Voxloud.com/whatsapp-automation/wa/templates?apiKey=*API KEY*
```

#### **Request Headers**

| **Header** | **Details** |
| ---------- | ----------- |
| apiKey     | API KEY     |

**Query Params**

| **Header** | <br />**Details** |
| ---------- | ----------------- |
| apiKey     | <br />API KEY     |

#### **Examples**

**Request**

```
curl --location --request GET 'https://api.v7. Voxloud.com/whatsapp-automation/wa/templates?apiKey=*API KEY*' \\
--header 'apiKey: *API KEY*'
```
