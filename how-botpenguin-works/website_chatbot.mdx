To create your Website chatbot , you first need to visit the  Cube6 Dashboard

![5 Website Chatbot Cube6 Resource Centre 1 Pn](/images/how-botpenguin-works/create-newchatbot/5.website_chatbot__Cube6_resource_centre-1.png)

To start creating your new chatbot, click **"Create New Bot."**

Select the primary purpose for creating your bot and proceed further.

![5 Website Chatbot Cube6 Resource Centre 2 Pn](/images/how-botpenguin-works/create-newchatbot/5.website_chatbot__Cube6_resource_centre-2.png)

Select the platform: “**Website Chatbot**”

![5 Website Chatbot Cube6 Resource Centre 3 Pn](/images/how-botpenguin-works/create-newchatbot/5.website_chatbot__Cube6_resource_centre-3.png)

You will see a notification: "**Bot has been successfully created.**”
