## **Customize the landing page**

1. Click on bots and select a website bot.
2. Click on "Website Chatbot Settings".

   ![50 Customize Landing Page Bot  Voxloud Resource Ce 1 Pn](/images/how-botpenguin-works/website-design/50.customize_landing_page_bot__Voxloud_resource_ce-1.png)
3. Under "Design" customize your bot and click on "Landing Page" to customize the same.

   ![50 Customize Landing Page Bot  Voxloud Resource Ce 2 Pn](/images/how-botpenguin-works/website-design/50.customize_landing_page_bot__Voxloud_resource_ce-2.png)

## **Content Section**

1. Enter your bot's landing page name in "Landing Page Name" box.
2. Enter the heading text you want to display in "Landing Page Header" box.
3. Enter the description text you want to display in "Landing Page Description" box.
4. Toggle ON/OFF to hide logo and text.
5. Toggle ON/OFF to hide chatbot's header.
6. Upload profile logo for your chatbot page.
7. Add any social media links and icons.
8. And click on "Save Changes" to update all the changes done.

   ![50 Customize Landing Page Bot  Voxloud Resource Ce 3 Pn](/images/how-botpenguin-works/website-design/50.customize_landing_page_bot__Voxloud_resource_ce-3.png)

## **Theme**

1. Select your chat window style from the given two options.
2. Select a bot icon from the options or upload from your device.
3. Select a landing page theme colour.
4. Select a chat background colour.
5. Select landing page background colour.
6. Select a font style for your chatbot.
7. Click on save changes to upload the changes done.

   ![50 Customize Landing Page Bot  Voxloud Resource Ce 4 Pn](/images/how-botpenguin-works/website-design/50.customize_landing_page_bot__Voxloud_resource_ce-4.png)

## **Layout**

Here you can select the layout and positioning for your landing page

Click on "Save Changes" to save your customizations.

![50 Customize Landing Page Bot  Voxloud Resource Ce 5 Pn](/images/how-botpenguin-works/website-design/50.customize_landing_page_bot__Voxloud_resource_ce-5.png)
