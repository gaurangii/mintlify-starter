<Accordion title="How can I update or change the error message being sent by the bot">
  You can customise the error messages by below steps:

  1. Open the chat flow builder and click on the chat component whose error message you want to change. 
  2. It will open up the option for that chat component on the right panel.
  3. Click on the "Advance" and Locate the error message field.
  4. Here you can change your error message as you wish.
