### **Get Segments/Groups**

This API endpoint allows you to retrieve segments for your  Voxloud application.

**Endpoint URL:** `https://api.v7. Voxloud.com/inbox/segments`

**HTTP Method:** GET

#### **Request Headers**

- `Authorization`: A bearer token is required for authentication. Replace `[MASKED_TOKEN]` with your valid API token. This token is used to authenticate and authorize the request.

#### **Sample Request**

```
curl 'https://api.v7. Voxloud.com/inbox/segments' \
  -H 'Authorization: Bearer [MASKED_TOKEN]' \
  -H 'Content-Type: application/json' \
  --compressed
```

Replace `[MASKED_TOKEN]` in the request with your actual API token.

#### **Response**

The API will respond with a JSON object containing the segments available in your  Voxloud application. The response format may vary depending on the segments you have configured.

Please replace `[MASKED_TOKEN]` in the request with your actual API token for actual usage.
