Voxloud website bots support all sort of websites. However most common of all is WordPress. Here is how you can integrate chatbots on WordPress websites.

### **1. Get your bot ready to go-live**

Before you move ahead, Make sure you have completed the following steps.

- Create your bot on the  Voxloud dashboard.
- Design your chatflow and other design settings from the bot setup page.
- Once Done, Go to the  Voxloud setup page and then click on the `Install your Chatbot.`
- Select the `WordPress` and copy the API Code/Key and save it safe with yourself.

### **2. Install the  Voxloud chat plugin from the WordPress plugin center**

Once you have completed the 1st step, Go to the WordPress site admin dashboard

- Find the WordPress plugin center and search for the ` Voxloud` in the search box.
- Once you find the  Voxloud Plugin, install the same.
- Upon installation, You will be able to see the `Activate` option. Click on the activate button to activate the plugin.

### **3. Paste API Code/key and make the bot live**

Once the  Voxloud plugin is activated, Look for the `Tools` in the side menu.

- In the side menu, Under tools, You will find the ` Voxloud` option.
- Open the ` Voxloud` option, and you will see the option to paste the  Voxloud API code / key which you have copied in 1st step.
- Paste the API key / Code, and proceed by saving the key.

### **4. Test your bot now**

You have completed all the heavy lifting.  Open your website in private mode (Incognito window) to avoid any cache issues and you will find  Voxloud chatbot on your website.
